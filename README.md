# 🏏 IPL Data Analysis Project

Welcome to my very first data analysis project! In this project, I used **Python** and the **Pandas** library to dig into a massive dataset containing the match history of **1,095 IPL matches across 17 seasons (from 2008 to 2024)**.

## 📊 Key Insights Uncovered
* **Toss Decisions:** Most teams preferred to field first after winning the toss to take advantage of the chasing conditions.
* **Consistency:** The most successful teams showed a high win percentage both at their home grounds and away matches.

## 🏆 Top 5 Most Successful IPL Teams
Using data aggregation, I calculated the total number of match wins for each franchise:

1. **Mumbai Indians** — 144 Wins 🏆
2. **Chennai Super Kings** — 138 Wins 🏆
3. **Kolkata Knight Riders** — 131 Wins 🏆
4. **Royal Challengers Bangalore** — 116 Wins 🏆
5. **Rajasthan Royals** — 112 Wins 🏆

## 🛠️ Tools & Technologies Used
* **Python** — For writing the core data analysis logic.
* **Pandas** — Used for data cleaning, filtering, and aggregation.
* **Jupyter Notebook** — The environment where the analysis was executed.
